% export/shim.tex — prepended to a reading unit before pandoc so it can digest
% siunitx units and neutralize \ensuremath. Document-agnostic: it hardcodes NO
% adopter preamble. Custom math macros, if the adopter has a preamble/macros.tex
% (or macros.tex), are appended after this shim by chapter-html.sh. Glossaries

% ============================================================================
